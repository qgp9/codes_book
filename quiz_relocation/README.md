# Relocation of array

## Question
```
If [a1,a2,a3...,an,b1,b2...bn] is given input change this to [a1,b1,a2,b2.....an,bn] , solution should be in-place
```

## Solution
* Time Complexity : O(logN)
* Space Complexity : O(logN)
